Intro

Consider this table of the English Premier League Standings taken from Sky Sports website:

The basic type of html table is simple, with little functionality. That's why I decided to make the BetterTables.js library, a library designed with simplicity and usability in mind for both the developer and the end user

Initializing The Table

To convert a basic existing HTML table into a BetterTable, use this line of code: let myBetterTable = new BetterTable(elementId = "initialization");, where elementId is the elementID of the table

Adding Rows/Columns

The BetterTable is capable of adding additional rows and columns to an existing table. Ensure when adding columns, the table has not been sorted. Rows can be inserted as such: BetterTable.addRow([column1Element, column2Element, column3Element,...]) Suppose we wanted to add an additional team to the Premier League Standings table, we would run MyBetterTable.addRow(["11", "My New Team", 30, 15, 4, 4, 30, 20, 10, 42]) Ensure the array is sorted in the same order as the column headers!

What if we wanted to add a new column to this existing table, ranking your favourite teams? It's a very similar process to adding a row, just insure the row values are in the correct order you'd like to insert them, and have a column header as your first argument. MyBetterTable.addColumn("Fav Teams", [5, 2, 4, 3, 6, 7, 9, 10, 11, 1])

Sorting By Column

This table is currently sorted by the column "#", which is a team's rank in the standings. Now I want to be able to sort the table by a different column? Maybe by my favourite teams ranking? Run this command: MyBetterTable.sortableHeaders() Now you can click on the headers to sort this table! If clicking on a header twice, table will toggle between ascending and descending sort
