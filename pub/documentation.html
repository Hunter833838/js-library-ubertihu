Documentation

Note

All of the documentation tables below are BetterTables, hope this helps absorb the information better!

Args Input Parameter

All of the fields specified in args are optional, as well as the args parameter itself

Field Type Default Value Description
sortable boolean false Enable sorting table by a specific column by clicking on it's header
exportToCSV boolean false Show button that allows user to download table contents as a CSV file
reorderColumns boolean false Enable reordering table columns by dragging it's header
search JSON None Show search bar that allows users to search the contents of search.header
search.header String Required if search Name of the column that the search bar will use to search through
tableData JSON None Will create new table (instead of parsing data from existing table) from TableData
tableData.headers array of strings Required if search List of headers to generate new table with
tableData.rows array of arrays of strings Required if search List of rows to generate new table with, rows[n].length must equal tableData.headers.length

BetterTable Functions

To see these functions in action, refer to the examples page!

Name Return Description
BetterTable(table, args) BetterTable Object Renders a BetterTable from table, and uses arguments in args
BetterTable.tableOrID(table) None Checks if table is a node representing a table, or a string representing element ID. If type is string, set internal elementID value. If table doesn't have an elementID, randomly generate one starting with "BT", then set internal elementID value
BetterTable.parseTableContents(String elementID) {data:{headers, rows}} Parses the content of table at element ID, returns content in json where headers = [header1, header2...] and rows = [[cell00,cell01...], [cell11, cell12]], also sets internal variables, headers and rows
BetterTable.renderTable() None Renders a BetterTable from id elementID, headers and rows (internal vars)
BetterTable.addSearchBar(String columnHeader) None Adds a search bar to the BetterTable, with id="searchBar"+elementID which searches through column columnHeader
BetterTable.addExportCSVButton() None Adds a button with id="exportButton"+elementID to export the contents of a table to CSV
BetterTable.addReorderableColumns() None Enables dragging of column headers to reorder them

BetterTable Properties

Default values are assuming we don't use the args input parameter, which sets these values

Name Type Default Description
headers array [] Stores the headers of the Table.
rows array of arrays [[]] Stores each row of the table in a seperate array. Length of each array must be equal to length of headers
elementID string "" The element of the table
sortable boolean false Allow sorting rows by clicking on a column header
exportToCSV boolean false Show a button to export the contents of a table to CSV
reorderColumns boolean false Allow dragging of column headers to reorder them

CSS Styling Override

To override the default CSS for a BetterTable, style the following properties

Fun fact: These documentation tables override the default styling!

CSS Element Description
.BTButton Export To CSV Button
.BTSearch BetterTable Search Bar
.BetterTable The Table itself
.BetterTable thead The head of the BetterTable
.BetterTable thead tr The header row of the BetterTable
.BetterTable th The header cells of the BetterTable
.BetterTable th.hoveringOver Appearance when a table header is being dragged over another table header
.BetterTable th.beingDragged Appearance when a table header is being dragged
.BetterTable tbody The body of the BetterTable
.BetterTable td The row cells of the BetterTable
